# 🖼️ Batch Image Resizer Tool (Python + Pillow)

## 📌 Project Overview
This project is a Batch Image Processing Tool built using Python and Pillow.  
It automatically resizes, converts, compresses, and optionally watermarks images inside a folder.

The tool processes all images from an input directory and saves optimized images into an output directory.


## 🚀 Features

- Resize images in batch
- Maintain aspect ratio
- Convert image formats (JPG, PNG, WEBP)
- Adjust image quality
- Add watermark text
- Auto-create output folder
- Clean and automated folder processing


## 🛠 Technologies Used

- Python 3
- Pillow (PIL)
- OS module
